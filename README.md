# ATM Simulator

A GUI-based ATM Simulator built using Java Swing.

## Features
- Account signup and login with PIN verification
- Deposit, withdraw, and balance enquiry
- Mini statement showing recent transactions
- Custom exception handling for insufficient balance
- Data persistence using file storage (accounts.txt, transactions.txt)

## Tech Stack
Java, Swing (GUI), OOP, Exception Handling, File I/O

## How to Run
1. Clone the repo
2. Compile: `javac *.java`
3. Run: `java ATMSimulator`
